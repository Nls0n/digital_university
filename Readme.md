### Бот помощник-цифровой университет c Mini-app


# Основной функционал
- Получение расписания
- Оформление отпусков
- Запись на день открытых дверей
- Подача заявок на академ отпуск
- Подача заявлений на поступление в ВУЗ
- Администрирование студентами
- Разделение ролей
- Общая информация
- Доступ к проектам организованным университетом
- Оплата обучения
- Подача заявлений на больничный


## Архитектура
Архитектура решения представлена монолитной архитектурой, с возможностью разделения на микросервисы.
Основной технологический стэк:
- Python 3.13
- FastAPI
- SQLAlchemy
- Requests
- Docker
- Docker-compose
- PostgreSQL (асинхронное взаимодействие - asyncpg)
- Redis (асинхронное кэширование - asyncio-redis)

Благодаря выбору современных инструментов и подходов к проектированию для оптимизации работы приложения мы 
имеем возможность для масштабирования функционала приложения, устойчивости к нагрузкам

В нашем решении в docker-образе разворачивается только бот, backend, frontend, PostgreSQL, Redis 
развернуты в облачных сервисах, для упрощения и ускорения сборки сервиса а также взаимодействия между 
бэкендом и фронтендом и ботом.


## Для запуска сервиса:
# Настройте .env (.env.prod)
# Запустите сборку докер контейнера
```cmd
docker-compose up --build
```
